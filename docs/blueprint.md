# **App Name**: FinQuiz Challenge

## Core Features:

- Quiz Initialization: Start the quiz after user enters their name and pays the entry fee.
- Question Display: Display a random question from a predefined set of financial questions, with multiple-choice answers.
- Answer Submission and Validation: Allow users to select an answer and validate it against the correct answer, updating the score accordingly.
- Timer Management: Implement a countdown timer for each question, which automatically fails the question if time runs out.
- Score Tracking: Keep track of the user's score throughout the quiz and display the final score at the end.
- Tab Switch Penalty: Implement a penalty for switching tabs during the quiz to discourage cheating.
- Adaptive Difficulty Adjustment: Employ an AI tool to evaluate the user's ongoing performance and adjust the quiz difficulty accordingly. Use a simple approach, like selecting easier or harder questions based on recent correct/incorrect answers. 

## Style Guidelines:

- Primary color: Deep blue (#2E3192) to reflect the stability and seriousness of financial matters.
- Background color: Very light blue (#E6E9F5), offering a clean, distraction-free backdrop.
- Accent color: Yellow (#FFC857), which creates strong contrast, highlights key interactive elements, and suggests optimism.
- Body and headline font: 'Inter', a versatile sans-serif font suitable for both headlines and body text due to its modern and clean appearance.
- Use financial and time-related icons throughout the app to enhance the user experience. All iconography should match in style with simple, clean lines.
- A clean, single-column layout is applied with questions and answers, ensuring focus and readability. It must include enough spacing around all the items to avoid a crammed look.
- Employ subtle animations for correct answer selections and transitions between questions to add engagement.
